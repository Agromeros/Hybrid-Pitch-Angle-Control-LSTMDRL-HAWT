## Introduction
Thesis project to obtain the Bachelor of Science (B.S) in Industrial Engineering with a specialization in Data Science.

It consists of the design of an intelligent hybrid system for wind turbines that combines wind speed prediction, using Deep Learning, and pitch angle control, using the Deep Reinforcement Learning paradigm. In addition, the control seeks to optimize the trade-off between rewards for maximizing captured energy and minimizing loads on blades and rotors.

The main objective of this project is to design a wind turbine pitching control system using deep learning & deep reinforcement learning.
